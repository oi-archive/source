 # 题目描述 
<p>
很久以前，三维世界还没有出现的时候，一个矩形的部落在这个世界上快乐的生活着。部落中的矩形的边长都平行于坐标轴。<br>某天，一个正方体突然闯入了这个世界，向矩形炫耀着他锐利的棱角和硕大的体积。所有的矩形敬畏的看着他，梦想着变成长方体。于是这个世界很快就变得完全不同了，所有的矩形开始竞相增大他们的面积、周长，甚至是对别的矩形的长宽比进行攻击。<br>很快，这场争斗演变为各自领域的抢夺。这个时候，任何一对矩形，如果他们占有相同的点（哪怕是周长上的一个点），他们都会变成敌人。<br>现在，你需要在这个社会中重建和平。你决定和任何一对成为敌人的矩形进行一次谈话，消除他们之间的怨恨。但是你需要矩形多少次这样的谈话呢？<br></p> 

 
 # 输入格式 
<p>
输入文件中有一个正整数N，表示矩形的数目。<br>以下N行，每行6个整数，描述一个矩形。前三个整数描述矩形一个角的坐标，而后三个整数描述这个矩形对角的坐标。<br>注意到每个矩形的边长都平行于某个坐标轴。这意味着对于每个矩形来说，其对角的两个顶点的坐标中至少有一维是相同的。<br></p> 

 
 # 输出格式 
<p>
你需要向输出文件写入一个整数，即有多少对矩形已经成为敌人。<br></p> 

 
 # 提示 
<p>
1 <= N <= 100 000<br>所有坐标在[1, 999]的范围内。<br></p>
