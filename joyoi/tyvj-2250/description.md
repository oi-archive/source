 # 题目描述 
<p>
w*h网格上有一些（最多3种）小写字母（鬼）,分别移动它们对应的大写字母里.每步可以有多个鬼同时移动,但每步结束之后任何两个鬼不能占用同一个位置,也不能在一步之内交换位置.所有空格连通,所有障碍格连通.任何一个2*2子网格至少有一个障碍格</p> 

 
 # 输入格式 
<p>
第一行给出W,H,N.<br>下面W行H列给出字符矩阵.</p> 

 
 # 输出格式 
<p>
最少需要的步数</p>
