 # 题目背景 
随着四叶草魔杖的插入，Nescafe之塔的塔尖附近出现了光漩涡。随着一束光芒投下，探险队员们面前出现了圣主的身影。<br>“参见圣主！”<br>“大家好，我叫杜宇飞。我就是Nescafe的圣主applepi。”<br>“圣主，我们是专程前来拜访您的！”探险队员们万分激动。<br>“再有七天就是Nescafe镇塔之宝一年一度的封印之期了，你们可以留下来观看封印。不过在这之前，我还是想考考你们……”<br><br><br> 

 
 # 题目描述 
若对于二叉树T的每个节点v，其左子树的高度L和右子树的高度R均满足|L&nbsp;–&nbsp;R|≤1，则这个树T有可能来自超自然之界。规定若某节点子树为空，则该子树的高度是0。你的任务是求有N个节点的可能来自超自然之界的树的数目。<br><br><br> 

 
 # 输入格式 
每个测试点包含若干个测试数据。<br>每个测试数据占一行，包含一个整数N。<br>输入文件以0结尾。<br><br><br> 

 
 # 输出格式 
对于每个测试数据，在单独的一行内输出结果。由于结果可能会很大，你只需要输出答案在十进制表示下的后九位。若答案不足九位，只需输出原答案。<br><br><br> 

 
 # 提示 
对于30%&nbsp;的测试点，N≤100。<br>对于70%&nbsp;的测试点，N≤1000。<br>对于100%&nbsp;的测试点，1≤N≤3000。<br><br><br>
