 # 题目描述 
<p>
考虑X-Y平面上满足X≥0的整数结点<br><br>给定若干个矩形及一个目标点，紧靠每个矩形外部的一周范围内没有其它矩形的点，所有矩形中的所有点满足X≥0<br><br>求一条以(0,0)点为起点，以目标点为终点、不与任何矩形相交的最短路径</p> 

 
 # 输入格式 
<p>
第一行输入为两个整数X与Y(1≤X≤106,-106≤Y≤106)表示目标点坐标<br><br>第二行为一个整数N(0≤N≤105)，表示矩形的数目<br><br>接下来的N行每行包括4个整数：<br>X1,Y1,X2,Y2(0≤X1,X2≤106,0≤Y1,Y2≤106)<br>即N个矩形的某对对角顶点。</p> 

 
 # 输出格式 
<p>
第一行包括一个整数L，表示最短路径长度<br></p>
