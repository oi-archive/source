 # 题目描述 
&nbsp;&nbsp;有n种硬币，面值为别为a[1],a[2],a[3]……a[n]，每种都有无限多。给定非负整数s，可以选取多少个硬币使得面值和恰好为s？输出硬币数目最小值和最大值 

 
 # 输入格式 
第1行n<BR>第2行s<BR>第3到n+2行为n种不同的面值 

 
 # 输出格式 
第1行为最小值<BR>第2行为最大值 

 
 # 提示 
1&lt;=n&lt;=100<BR>1&lt;=s&lt;=10000<BR>1&lt;=a[i]&lt;=s
