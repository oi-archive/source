 # 题目描述 
<p>
<br>卡卡开始养兔子了！妈妈给他买了一对刚出生的兔子，卡卡了解到兔子的繁殖规律是这样的：才出生的一对兔子在一个月后将第一次生出一胎a对兔子，接着在出生后的二个月又将生出b对兔子，在第三个月和以后每个月都会繁殖c对兔子。(a <= b <= c)<br>由斐波纳契数列我们知道兔子的繁殖速度是很快的，然而卡卡有兔子一样多的好朋友，卡卡想在m个月后有k对兔子，以便分给他们的好友，他的愿望是否能够实现呢？<br>[任务]<br>编写一个程序：<br>&#61548;	从输入文件中读入输入信息；<br>&#61548;	计算m个月后卡卡将有多少对兔子，设之为P；<br>&#61548;	计算如果m个月后卡卡要拥有至少k对兔子，那么开始时妈妈至少应该为卡卡购买多少对兔子，设之为Q；<br>&#61548;	将结果输出至输出文件。<br></p> 

 
 # 输入格式 
<p>
输入文件的第一行有4个正整数：a, b, c和m；而第二行则仅含一个正整数k。它们的含义见上文描述。<br></p> 

 
 # 输出格式 
<p>
你的程序将向输出文件输出两行，第一行是一个整数P而第二行是一个整数Q。<br></p>
