 # 题目描述 
<p>
<br>今天是hidadz小朋友的生日，她邀请了许多朋友来参加她的生日party。 hidadz带着朋友们来到花园中，打算坐成一排玩游戏。为了游戏不至于无聊，就座的方案应满足如下条件：<br>对于任意连续的一段，男孩与女孩的数目之差不超过k。<br>很快，小朋友便找到了一种方案坐了下来开始游戏。hidadz的好朋友Susie发现，这样的就座方案其实是很多的，所以大家很快就找到了一种，那么到底有多少种呢？热爱数学的hidadz和她的朋友们开始思考这个问题……<br>假设参加party的人中共有n个男孩与m个女孩，你是否能解答Susie和hidadz的疑问呢？由于这个数目可能很多，他们只想知道这个数目除以12345678的余数。<br></p> 

 
 # 输入格式 
<p>
仅包含一行共3个整数，分别为男孩数目n, 女孩数目m,  常数k。<br></p> 

 
 # 输出格式 
<p>
应包含一行，为题中要求的答案。<br></p> 

 
 # 提示 
<p>
对于30%的数据，n , m ≤ 20；<br>对于100%的数据, n , m ≤ 150，k ≤ 20。<br></p>
