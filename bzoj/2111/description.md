
# Description

<div class="content"><p>称一个1,2,...,N的排列P1,P2...,Pn是Magic的，当且仅当2&lt;=i&lt;=N时，Pi&gt;Pi/2. 计算1，2，...N的排列中有多少是Magic的，答案可能很大，只能输出模P以后的值</p></div>

# Input

<div class="content"><p>输入文件的第一行包含两个整数 n和p，含义如上所述。</p></div>

# Output

<div class="content"><p>输出文件中仅包含一个整数，表示计算1,2,⋯, 的排列中， Magic排列的个数模 p的值。</p></div>

# Sample Input

<div class="content"><span class="sampledata">20 23 </span></div>

# Sample Output

<div class="content"><span class="sampledata">16 </span></div>

# Hint

<div class="content"><p></p><p>100%的数据中，1 ≤  N ≤ 106, P ≤ 10^9，p是一个质数。 <span style="color: #ff0000">数据有所加强</span></p><p></p></div>
