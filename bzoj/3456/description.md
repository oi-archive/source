
# Description

<div class="content"><p><span style="font-size: medium"> 刚刚解决完电力网络的问题, 阿狸又被领导的任务给难住了.<br/>
 刚才说过, 阿狸的国家有n个城市, 现在国家需要在某些城市对之间建立一些贸易路线, 使得整个国家的任意两个城市都直接或间接的连通. 为了省钱, 每两个城市之间最多只能有一条直接的贸易路径. 对于两个建立路线的方案, 如果存在一个城市对, 在两个方案中是否建立路线不一样, 那么这两个方案就是不同的, 否则就是相同的. 现在你需要求出一共有多少不同的方案.<br/>
 好了, 这就是困扰阿狸的问题. 换句话说, 你需要求出n个点的简单(无重边无自环)无向连通图数目.<br/>
 由于这个数字可能非常大, 你只需要输出方案数mod 1004535809(479 * 2 ^ 21 + 1)即可.<br/>
</span></p></div>

# Input

<div class="content"><p><font size="4"> 仅一行一个整数n(&lt;=130000)<br/>
 <br/>
</font></p></div>

# Output

<div class="content"><p><font size="4"> 仅一行一个整数, 为方案数 mod 1004535809.<br/>
 <br/>
</font></p></div>

# Sample Input

<div class="content"><span class="sampledata"> 3<br/>
<br/>
</span></div>

<div class="content"><span class="sampledata"> 4<br/>
</span></div>
