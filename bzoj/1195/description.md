
# Description

<div class="content"><p>给定n个字符串（S1,S2,„,Sn），要求找到一个最短的字符串T，使得这n个字符串（S1,S2,„,Sn）都是T的子串。</p></div>

# Input

<div class="content"><div>第一行是一个正整数n（n&lt;=12），表示给定的字符串的个数。</div>
<div>以下的n行，每行有一个全由大写字母组成的字符串。每个字符串的长度不超过50.</div></div>

# Output

<div class="content"><div>只有一行，为找到的最短的字符串T。在保证最短的前提下，</div>
<div>如果有多个字符串都满足要求，那么必须输出按字典序排列的第一个。</div></div>

# Sample Input

<div class="content"><span class="sampledata">2 <br/>
ABCD<br/>
BCDABC</span></div>

# Sample Output

<div class="content"><span class="sampledata">ABCDABC</span></div>
