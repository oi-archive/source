
# 题目描述

**原题来自：USACO 2003 Mar. Green**

给定一个长度为 $n$ 的非负整数序列 $A$ ，求一个平均数最大的，长度不小于 $L$ 的子段。

# 输入格式

第一行用空格分隔的两个整数 $n$ 和 $L$；

第二行为 $n$ 个用空格隔开的非负整数，表示 $A_i$。

# 输出格式

输出一个整数，表示这个平均数的 $1000$ 倍。不用四舍五入，直接输出。

# 样例

#### 样例输入
```plain
10 6
6 4 2 10 3 8 5 9 4 1 
```

#### 样例输出
```plain
6500
```

# 数据范围与提示

$1 \leq n \leq 10^5,0 \leq A_i \leq 2000$。
