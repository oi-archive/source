
# 题目描述

这是一道模板题。

给定一个字符串 $ A $ 和一个字符串 $ B $，求 $ B $ 在 $ A $ 中的出现次数。$A$ 和 $B$ 中的字符均为英语大写字母或小写字母。

$ A $ 中不同位置出现的 $ B $ 可重叠。

# 输入格式

输入共两行，分别是字符串 $ A $ 和字符串 $ B $。

# 输出格式

输出一个整数，表示 $ B $ 在 $ A $ 中的出现次数。

# 样例

#### 样例输入
```plain
zyzyzyz
zyz
```

#### 样例输出
```plain
3
```

# 数据范围与提示

$ 1 \leq  A, B$ 的长度 $ \leq 10 ^ 6 $，$ A $、$ B $ 仅包含大小写字母。
