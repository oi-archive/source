
# 题目描述

给出一个简单无向图，判断是否存在两个长度相同的简单环。

# 输入格式

第一行两个正整数 $n$ , $m​$ 表示点数和边数。

接下来 $m$ 行每行两个正整数描述一条边。

# 输出格式

若存在，输出 `Yes`；否则，输出 `No`。

# 样例


#### 样例输入 1

```
10000 0
```

#### 样例输出 1

```
No
```

#### 样例输入 2

```
5 6
1 2
2 3
3 1
1 4
4 5
5 1
```

#### 样例输出 2

```
Yes
```

# 数据范围与提示


对于所有测试数据，$1\le n \le 10^4$，$1\le m \le 10^6$。


| 子任务编号  | $n\le$ | 分值 |
| :--------: | :-----: | :----------: |
| $1$          | $10$    | $40$ |
| $2$          | $20$    | $20$ |
| $3$          | $400$ | $20$ |
| $4$          | $10000$ | $20$ |
