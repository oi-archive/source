
# 题目描述

**来源：SDOI 2009**  

Sheng Bill 有着惊人的心算能力，甚至能用大脑计算出两个巨大的数的 GCD（最大公约数）！因此他经常和别人比赛计算 GCD。有一天 Sheng Bill 很嚣张地找到了你，并要求和你比赛，但是输给 Sheng Bill 岂不是很丢脸！所以你决定写一个程序来教训他。

# 输入格式

输入共两行，第一行一个数 $A$，第二行一个数 $B$。

# 输出格式

一行，表示 $A$ 和 $B$ 的最大公约数。

# 样例

#### 样例输入
```plain
12
54
```

#### 样例输出
```plain
6
```


# 数据范围与提示

对于全部数据，$0\lt A,B\le 10^{10000}$。
