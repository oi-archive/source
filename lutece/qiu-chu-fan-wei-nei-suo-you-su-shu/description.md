
# Content

给出整数$a,b$，求出区间$[a,b]$内的所有素数。

# Standard Input

有多组测试数据。输入的第一行是整数$T$（$0<T\le 1000$），表示测试数据的组数。每一组测试数据只有一行，分别为整数$a$和$b$，两数之间有一个空格。该行没有其它多余的符号。$0<a<b<2^{16}$。

# Standard Output

对应每组输入，输出一行$[a,b]$区间内所有的素数，每个数后应有一个空格，如果$[a,b]$区间内没有素数，则该行为空行。该行不能有其它多余的符号。
