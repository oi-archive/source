
# Content

张家有三姐妹:`兑田`，`兄田`，`儿田`分别高$A$ cm,$B$ cm,$C$ cm，有天一个神秘商人向她们兜售神仙水，传说擦了神仙水可以长高高。  
每瓶神仙水的容量有限，可以给一个姐妹涂长高2cm或者给两个姐妹涂各长高1cm，三姐妹想长着一样高，但是由于还要买裙子，所以她们想花尽量少的钱，同样是女孩子的你可以帮帮她们吗。

# Standard Input

$0<=A,B,C<=10^{8}$    
注意时间限制，单纯的模拟可能会导致TLE。

# Standard Output

输出一个数表示至少要买多少瓶。
