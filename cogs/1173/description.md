# 题目描述


超空间电台<br/>
hyper<br/>
【问题描述】<br/>
Henryy决定在宇宙中建立超空间电台。但是在宇宙中所有的行星具有两种意识形态之一。一种是环保形的，另一种是经济形的。Henryy广播的内容，若是环保形的，这样环保形行星不愿意收听；若是经济形的，这样经济形行星不愿意收听（或许彼此觉得对方的内容新鲜吧）。宇宙广播管理局对超空间电台的建立有严格的审查，如果要在某个行星申请以R为半径的超空间广播电台，必须满足下列条件：<br/>
在A行星上必须广播与A行星意识形态相同的内容。<br/>
NA+表示在A行星上面以R为半径的广播中，可以广播到的与A行星意识形态相同的行星（包括A自己）数量。<br/>
NA-表示在A行星上面以R为半径的广播中，可以广播到的与A行星意识形态不相同的行星数量。<br/>
如果要通过申请，则要求NA+<na-。<br>
由于购买以R为半径的广播器材的前期设计费用不菲，所以不可能订购多种半径范围的广播器材。因此Henryy想知道应该设计半径为多少的广播器材，才能使他能够在尽可能多的行星上面建立广播电台。如果有多个R合乎要求，则要求R尽量小，因为大范围广播不单只浪费能源，而且是一种污染。<br/>
你的任务就是帮助Henryy找到这个半径R。<br/>
【输入文件】<br/>
第一行有一个整数N，表示有多少个行星。接下来的N行每行有四个数Xi，Yi，Zi，Di。Xi，Yi，Zi表示第i个行星的坐标位置，Di表示第i个行星的意思形态类型，Di只可能是0，1之一。坐标绝对值不超过10000。<br/>
【输出文件】<br/>
第一行输出一个整数M，表示最多可以通过M个行星的申请。<br/>
第二行输出一个浮点数R（保留到小数点后4位），表示达到最大申请数时的最小广播范围。<br/>
【样例输入】<br/>
4<br/>
0 0 1 0<br/>
1 0 0 0<br/>
1 0 1 1<br/>
0 0 0 1<br/>
【样例输出】<br/>
4<br/>
1.0000<br/>
【数据约定】<br/>
对于40%的数据，有N&lt;=30；<br/>
对于100%的数据，有N&lt;=500。<br/>
<br/>
</na-。<br>
